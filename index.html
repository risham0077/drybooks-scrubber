Using a Shared Worker (no key needed)

  1. Your admin (the person who set this up) should have sent you a Worker URL — it looks like https://drybooks-scrubber.something.workers.dev.
  2. Paste that URL into the Worker URL box above.
  3. If your admin also set a shared password, paste it into the Shared Password box. Otherwise leave it blank.
  4. Click Save. You're done — drop a PDF on the home screen to test.
If you're the admin setting this up: See cloudflare-worker.js and the README in the repo. Briefly: create a free Cloudflare account, paste the Worker script, set your Gemini key as an encrypted variable named GEMINI_API_KEY, click Deploy, copy the URL, share it with your team. ~15 minutes one-time, $0/month.

How to get a free Gemini API key

  1. Open aistudio.google.com/app/apikey in a new tab.
  2. Sign in with any Google account (the one you use for Gmail is fine).
  3. If prompted, accept the Google AI Studio terms of service.
  4. Click the blue Create API key button on the right.
  5. Choose Create API key in new project if asked (or pick any existing project).
  6. Your key will appear — it starts with AIza and is about 39 characters long. Click the copy icon next to it.
  7. Come back to this tab and paste it into the Your API Key box above.
  8. Click Save. You're done — drop a PDF on the home screen to test it.
Free tier covers 1,500 scrubs per day. No credit card required. The key only works with Gemini — keep it secret like a password.

How to get an Anthropic Claude API key

  1. Open console.anthropic.com in a new tab.
  2. Click Sign up (or sign in if you already have an account). New accounts get $5 in free credit.
  3. Verify your email and phone number when prompted.
  4. Once in the console, click Settings in the left sidebar, then API keys.
  5. Click Create Key. Give it a name like DryBooks Scrubber.
  6. Copy the key shown — it starts with sk-ant-. You won't be able to see it again, so copy it now.
  7. Come back to this tab and paste it into the Your API Key box above.
  8. Click Save.
$5 free credit ≈ 50 report scrubs. After that, expect $0.05–$0.20 per report. Quality is noticeably better than Gemini for nuanced narrative review.

How to get an OpenAI API key

  1. Open platform.openai.com/api-keys in a new tab.
  2. Sign up or sign in. Note: OpenAI requires you to add a payment method before you can create keys.
  3. Go to Billing in the left sidebar and add a payment method. You can set a hard spend cap (recommended: $5/month to start).
  4. Back on the API keys page, click Create new secret key.
  5. Give it a name like DryBooks Scrubber and click Create.
  6. Copy the key shown — it starts with sk-. You can't see it again later.
  7. Come back to this tab and paste it into the Your API Key box above.
  8. Click Save.
Costs ~$0.05 per report scrub on GPT-4o-mini. Set a monthly spend cap in OpenAI's billing settings to avoid surprises.

Running without AI

  1. You don't need to enter a key for this mode.
  2. Click Save to confirm.
  3. The scrubber will still run all hard rules — timestamps, dehu sizing, missed readings, fabrication signals, etc.
  4. You lose the AI judgment layer: Class/Category recommendations, nuanced narrative review, and corrective-action suggestions.
Use this if you're testing the tool, working offline, or restricted from cloud AI services. You can switch back to a provider anytime.
